## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| worker→modal dialog | grab_set + wait_window 必须仅在主线程 |
| 多 Toplevel grab 顺序 | transaction + continue 叠加导致死锁 |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-01-03 | Denial of Service | grab_set 死锁 | mitigate | _prepare_for_continue_dialog 释放/销毁 transaction_window |
| T-01-04 | Denial of Service | show_results messagebox 阻塞 worker | mitigate | run_on_main_thread + 非模态 result |
| T-01-05 | Spoofing | worker 伪造 UI 状态 | accept | 本地单用户桌面，无远程攻击面 |
| T-01-SC | Tampering | 无新包 | accept | 不引入新依赖 |
</threat_model>
